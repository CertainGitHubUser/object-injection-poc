# object-injection-poc

# POC that shows how to make object injection using phar and jpg polyglot file.

## Deploy
```sh
# Run following command in the project root
./set_up.sh
```

## Usage
```sh 
# To Create file with payload run:
./create_payload.sh
```

```sh 
# To test file with payload run: 
./test_payload.sh
```
